Web Development

What is it?

Web Development is what builds the websites and web applications that we use in our everyday lives. It involves everything from making a simple website to a mammoth social media platform.

Websites can be styled and designed as beautifully as you like. Check out Apple's iPhone SE webpage (and just keep scrolling...) to see what can be possible with web development!

How are webpages built?

Websites are created using three main building blocks:

  • HTML - the text and images on the page
  • CSS - how you want the text and images to look
  • Javascript - adding behaviour to your website to make it dynamic and interactive
