Get coding!


Step 1: Install Visual Studio Code

To start making your own website, a great start is getting a code editor which you can use to write your HTML.

Visual Studio Code (VSCode) is great for this. You can download it here.


Step 2: Install the Live Server plugin

VSCode has a great plugin that allows you to see your webpage in action as you save changes without having to reload anything.

  • Open the Extensions section on the left-hand side of VSCode
  • Search for Live Server in the search bar and select the first result
  • Click Install

The extension will now be installed and ready for us to use.


Step 3: Create a folder on your computer and open in Visual Studio

  • Create a folder anywhere you like on your computer and name it e.g. learn-first-demo.
  • Open this folder in VSCode by going to File --> Open --> Select your folder

Step 4: Create an HTML file in VSCode

  • The left-hand side of VSCode will show your folder open (with no files).
  • Right-click in this space and select New File and name the file index.html.

Step 5: Let's add some content to our HTML page!

  • Copy the below code into your index.html file
<h1>Welcome to my page!</h1>
<p>Here are some of my favourite interests:</p>
<ul>
  <li>Cooking</li>
  <li>Reading</li>
  <li>Running</li>
</ul>
<p>Here is my favourite landmark in the world:</p>
<img src="https://upload.wikimedia.org/wikipedia/commons/e/e3/Kheops-Pyramid.jpg">

  • Save your file by navigating to File --> Save
  • You're all set to open your website!

Step 6: Open your webpage!

  • Right-click your file in the left-side pane and select Open with Live Server
  • This will automatically open your webpage in your browser!
