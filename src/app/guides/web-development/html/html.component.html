Intro to HTML

What is HTML?


HTML stands for Hyper Text Markup Language.

It is the skeleton structure of any webpage - any image, video or text that you see on a webpage has been defined in an HTML page.

How is HTML written?


Content is defined in HTML using tags. Below is a list of the main tags you can use to surround and define your content.

<p> - defines a paragraph

<p>This is a paragraph</p>

This is a paragraph

<h1> - defines a page title

<h1>Here is my page title</h1>

Here is my page title

<h2> - defines a sub-heading

<h2>Here is my sub-heading</h2>

Here is my sub-heading

Hover over me Tooltip text

<a> - defines a link to a URL / website

<a href="https://www.google.com">Here's a link to Google</a>

Pay attention to the href part - this is where you define the URL to link to

Here's a link to Google

<img> - defines an image either available online or on your computer

<img src="https://s1.cdn.autoevolution.com/images/news-pictures-600x/bugatti-chiron-production-ending-in-2021-fewer-than-100-units-still-available-140517-7.jpg">

Note 1: pay attention to the src part - this is where you define either the url or the file path to your image

Note 2: no closing tag needed!

Get coding!


Step 1: Install Visual Studio Code

To start making your own website, a great start is getting a code editor which you can use to write your HTML.

Visual Studio Code (VSCode) is great for this. You can download it here.


Step 2: Install the Live Server plugin

VSCode has a great plugin that allows you to see your webpage in action as you save changes without having to reload anything.

  • Open the Extensions section on the left-hand side of VSCode
  • Search for Live Server in the search bar and select the first result
  • Click Install

The extension will now be installed and ready for us to use.
